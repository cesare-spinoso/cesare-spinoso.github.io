Exercise. Show that the Halting Problem \(\textsf{HP}\) is undecidable by giving a reduction from the Acceptance Problem \(\textsf{AP}\).
  • \(\textsf{HP}:\) Given a TM \(M\) and an input \(x\), does \(M\) halt on \(x\)?
  • \(\textsf{AP}:\) Given a TM \(M\) and an input \(x\), does \(M\) accept \(x\)?
